# Smoke Control

Smoke Control is a project aimed at controlling smoke and fog machines via the remote control input using DMX. Cheap smoke machines sometimes come without any DMX input, but often have some kind of wired remote that usually uses female 3 pin XLR
![3 pin XLR](https://en.wikipedia.org/wiki/File:XLR_pinouts.svg)

We have found 2 ways different manufacturers use the XLR pins. ON trigger can be by connecting pins
- 1 - 3
- 2 - 3
